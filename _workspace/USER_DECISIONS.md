# 사용자 확정 결정 (2026-06-01) — 모든 에이전트 우선 반영

콘텐츠 브리프 상단 "정보 요청" 6항목에 대한 사용자 확정. 기존 placeholder/가정을 아래로 교체한다.

## 1. 아이디어 — 임시 작성 + 마인드맵 인터랙션 (중요)
- 구체 아이템을 **임시로 작성해 넣어도 됨** (확정 아님, "검토 중인 후보" 톤).
- **UX 요구(신규):** 아이디어 섹션을 **마인드맵 형식**으로 구성한다.
  - **중앙 허브 노드** = 팀의 방향("AI 활용 실사용·수익형 서비스").
  - 중앙에서 **여러 아이디어 가지가 뻗어 나간다**(방사형 브랜치).
  - **각 노드를 클릭하면** 그 아이디어의 구체 내용(문제/사용자/AI 적용/수익 모델 등)이 상세 패널·모달로 펼쳐진다.
  - 후보 아이디어는 팀 강점(AI 자동화·실서비스 운영·창업)에 어울리는 4~6개를 임시 제안. 각 노드에 1~3문장 상세.
  - 모바일에서도 동작해야 함(클릭/탭). 접근성: 키보드 포커스·aria.
